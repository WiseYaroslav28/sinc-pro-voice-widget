# ROADMAP SINC PRO

Пошаговый план доведения приложения до релиза.
Пользователь помечает задачи тегом `[VERIFIED]`, когда функция работает корректно. **Агентам ЗАПРЕЩАЕТСЯ ломать или переписывать `[VERIFIED]` код без прямого приказа.** При внесении изменений опираться на переиспользование компонентов.

## Этап 1: Стабилизация фундамента (Rust + WinAPI)
- `[x]` **DPI и Мультимониторность**: Реализовать команду `get_cursor_monitor` в `lib.rs`. `[VERIFIED]`
- `[x]` **Менеджер настроек**: Надежное хранение API-ключей Google Gemini и Yandex. `[VERIFIED]`
- `[x]` **Полировка интерфейса капсулы**: Динамические зоны охвата, прозрачность ОС-окна, hover-эффекты и плавность взаимодействий. Текущее состояние капсулы полностью устраивает. `[VERIFIED]`

## Этап 2: Механика умной диктовки (Rust + JS)
- `[x]` **Умные хоткеи (Ctrl+Win)**: `[VERIFIED]`
  - Однократное нажатие: пауза/старт.
  - Удержание (`Ctrl+W`): PTT-режим (запись, пока зажато).
  - Быстрое двойное нажатие: Lock-режим.
  - `Win+Alt`: Принудительная отправка (Force Send).
  - `Escape`: отмена записи с таймером 5 секунд.
- `[x]` **Накопитель аудио**: На бэкенде накапливать куски при паузах и склеивать перед отправкой. `[VERIFIED]`
- `[x]` **Звуковое оформление (Audio Cues)**: Системные щелчки (старт записи, стоп, пауза, отправка, отмена). `[VERIFIED]`
- `[x]` **Визуализатор голоса**: Интеграция Web Audio API (`AnalyserNode`) в капсуле. `[VERIFIED]`
- `[x]` **Волшебная палочка (Пресеты ИИ)**: Выбор действия после диктовки (Транскрипция, Summary, Кастомный промпт "Горя"). `[VERIFIED]`

## Этап 3: Механика Озвучки (TTS), Дашборд TTS и Безопасный Буфер
- `[x]` **Безопасный буфер (Rust)**: Паттерн Save-Copy-Restore в WinAPI. Анти-залипание модификаторов через `dwExtraInfo` фильтр + принудительный сброс атомарных флагов после `simulate_ctrl_c`. `[VERIFIED v3.4.0-beta]`
- `[x]` **Глобальные хоткеи озвучки**: `[VERIFIED v3.4.0-beta]`
  - `Ctrl+Shift`: Копирование текста и авто-чтение. Многократное срабатывание без F5.
  - `Ctrl+Alt`: Копирование текста, принудительный перевод и чтение.
  - Механика fire-and-forget: `SHORTCUT_ACTIVE` сбрасывается сразу после эмита.
- `[x]` **Раздел "Озвучка / Перевод" в Дашборде**: `[VERIFIED]`
  - Новая зона с редактируемым текстовым полем (textarea), куда прилетает скопированный текст.
  - Умная кнопка Play/Pause/Stop в Журнале ИИ с бесшовным CSS-мостиком для hover.
  - Подсветка фона под читаемыми предложениями, идеальная синхронизация скролла.
- `[x]` **Плавающий виджет озвучки**: Прозрачное окно с хитбоксами (`updateClickRegion` по паттерну капсулы). Сохранение настроек голоса/скорости в `localStorage`. `[VERIFIED v3.4.0-beta]`
- `[ ]` **Санитаризация Markdown**: Очистка текста от спецсимволов перед отправкой в Edge TTS.

## Этап 4: Оверлей распознавания (OCR) и Переводчик
- `[ ]` **Тумблер OCR**: Добавление тумблера в Дашборд (по аналогии с капсулой) для включения/выключения рамки захвата экрана (шорткат не назначаем).
- `[ ]` **Оверлей переводчика**: Lock Mode (блокирующий) и Passthrough Mode (прозрачный для кликов).
- `[ ]` **UI "Бусины на нити"**: Рендеринг переведенных предложений с соединительными линиями (динамический SVG), не перекрывая оригинал.
- `[ ]` **Сквозной TTS-пайплайн**: Автоматическая передача распознанного экранного текста в виджет "Озвучка / Перевод" в дашборде.

## Этап 5: Финализация Dashboard и Настроек
- `[ ]` **Главный Dashboard (Home)**: Общая статистика, тумблеры виджетов, статусы подключений API. Блок активности динамически тянется вниз, не перекрываясь футером.
- `[ ]` **Интерактивный Журнал ИИ**: Блоки текста с иконками аудио, переключателями (оригинал/summary) и сохранением состояния (состояние комбобоксов не сбрасывается при скролле).
- `[ ]` **Менеджер пресетов (Настройки)**: Возможность создавать, редактировать, удалять кастомные промпты.

## Этап 6: Подготовка к релизу, автообновления и защита от сбоев
- `[ ]` **Механизм автоматических обновлений (Tauri Updater)**: Настройка проверки версий, автоскачивания обновлений с GitHub Releases и обновление `docs/updater.json`.
- `[ ]` **Авто-обнаружение Edge TTS токена**: Текущий `TrustedClientToken` захардкожен (публичный, из браузера Edge). Если Microsoft его поменяет — озвучка сломается. Варианты решения:
  - **Парсинг из Edge**: При первом запуске искать установленный Edge, извлекать токен из его JS-файлов.
  - **Фоллбэк-сервер**: Микросервис, который периодически проверяет актуальность токена и раздаёт клиентам. Обновление в одном месте → все пользователи получают автоматически.
  - **Гибридный подход**: Сначала пробовать захардкоженный → при 403 пытаться извлечь из Edge → при неудаче запрашивать с сервера.
- `[ ]` **Сборка установщика (MSI/NSIS)**: Tauri bundle для Windows с автообновлением.
- `[ ]` **Автотесты перед релизом**: Playwright + встроенный self-test покрывают все критические пути (TTS, хоткеи, OCR).
