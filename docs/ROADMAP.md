## Этап 2: Механика умной диктовки (Rust + JS)
- `[x]` **Умные хоткеи (Ctrl+Win)**: `[VERIFIED]`
  - Однократное нажатие: пауза/старт.
  - Удержание (`Ctrl+W`): PTT-режим (запись, пока зажато).
  - Быстрое двойное нажатие: Lock-режим.
  - `Win+Alt`: Принудительная отправка (Force Send).
  - `Escape`: отмена записи с таймером 5 секунд.
- `[x]` **Накопитель аудио**: На бэкенде накапливать куски при паузах и склеивать перед отправкой. `[VERIFIED]`
- `[x]` **Звуковое оформление (Audio Cues)**: Системные щелчки (старт записи, стоп, пауза, отправка, отмена). `[VERIFIED]`
- `[x]` **Визуализатор голоса**: Интеграция Web Audio API (`AnalyserNode`) в капсуле. `[VERIFIED]`
- `[x]` **Волшебная палочка (Пресеты ИИ)**: Выбор действия после диктовки (Транскрипция, Summary, Кастомный промпт "Горя"). `[VERIFIED]`

## Этап 3: Механика Озвучки (TTS), Дашборд TTS и Безопасный Буфер
- `[ ]` **Безопасный буфер (Rust)**: Реализовать паттерн Save-Copy-Restore в WinAPI (с анти-залипанием модификаторов).
- `[ ]` **Глобальные хоткеи озвучки**: 
  - `Ctrl+Shift`: Копирование текста в дашборд и авто-чтение (или перевод, если включен тумблер на панели).
  - `Ctrl+Alt`: Копирование текста, принудительный перевод и чтение.
- `[x]` **Раздел "Озвучка / Перевод" в Дашборде**: `[VERIFIED]`
  - Новая зона с редактируемым текстовым полем (textarea), куда прилетает скопированный текст.
  - Умная кнопка Play/Pause/Stop в Журнале ИИ с бесшовным CSS-мостиком для hover.
  - Подсветка фона под читаемыми предложениями, идеальная синхронизация скролла.
- `[ ]` **Санитаризация Markdown**: Очистка текста от спецсимволов перед отправкой в Edge TTS.

## Этап 4: Оверлей распознавания (OCR) и Переводчик
- `[ ]` **Тумблер OCR**: Добавление тумблера в Дашборд (по аналогии с капсулой) для включения/выключения рамки захвата экрана (шорткат не назначаем).
- `[ ]` **Оверлей переводчика**: Lock Mode (блокирующий) и Passthrough Mode (прозрачный для кликов).
- `[ ]` **UI "Бусины на нити"**: Рендеринг переведенных предложений с соединительными линиями (динамический SVG), не перекрывая оригинал.
- `[ ]` **Сквозной TTS-пайплайн**: Автоматическая передача распознанного экранного текста в виджет "Озвучка / Перевод" в дашборде.

## Этап 5: Финализация Dashboard и Настроек
- `[ ]` **Главный Dashboard (Home)**: Общая статистика, тумблеры виджетов, статусы подключений API. Блок активности динамически тянется вниз, не перекрываясь футером.
- `[ ]` **Интерактивный Журнал ИИ**: Блоки текста с иконками аудио, переключателями (оригинал/summary) и сохранением состояния (состояние комбобоксов не сбрасывается при скролле).
- `[ ]` **Менеджер пресетов (Настройки)**: Возможность создавать, редактировать, удалять кастомные промпты.
