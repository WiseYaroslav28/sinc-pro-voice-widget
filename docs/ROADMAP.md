# ROADMAP SINC PRO

Пошаговый план доведения приложения до релиза.
Пользователь помечает задачи тегом `[VERIFIED]`, когда функция работает корректно. **Агентам ЗАПРЕЩАЕТСЯ ломать или переписывать `[VERIFIED]` код без прямого приказа.**

## Этап 1: Стабилизация фундамента (Rust + WinAPI)
- `[x]` **DPI и Мультимониторность**: Реализовать команду `get_cursor_monitor` в `lib.rs` (используя `GetCursorPos`, `MonitorFromPoint`, `GetDpiForMonitor`).
- `[x]` **Нативный буфер обмена**: Реализовать команды `read_clipboard_text` и `write_clipboard_text` в `lib.rs` для работы Edge TTS.
- `[x]` **Менеджер настроек**: Добавить надежное хранение API-ключей Google Gemini и Yandex, чтобы приложение не падало при их отсутствии.
- `[x]` **Полировка интерфейса капсулы**: Динамические зоны охвата, прозрачность ОС-окна, hover-эффекты и плавность взаимодействий.

## Этап 2: Механика умной диктовки (Rust + JS)
- `[x]` **Умные хоткеи (Ctrl+Win)**: `[VERIFIED]`
  - Однократное нажатие: пауза/старт.
  - Удержание (`Ctrl+W`): PTT-режим (запись, пока зажато).
  - Быстрое двойное нажатие: Lock-режим.
  - `Escape`: отмена с таймером 5 секунд.
- `[x]` **Накопитель аудио**: На бэкенде накапливать куски при паузах и склеивать перед отправкой. `[VERIFIED]`
- `[x]` **Звуковое оформление (Audio Cues)**: Системные щелчки (старт записи, стоп, пауза, отправка, отмена). `[VERIFIED]`
- `[x]` **Визуализатор голоса**: Интеграция Web Audio API (`AnalyserNode`) в капсуле. Волнистые линии при разговоре, прямая линия при тишине. `[VERIFIED]`
- `[ ]` **Волшебная палочка (Пресеты ИИ)**: Выбор действия после диктовки (Транскрипция, Summary, Кастомный промпт "Горя").

## Этап 3: Интеграция OCR и экранного переводчика
- `[ ]` **Выделение области и OCR**: Рамка захвата экрана + отправка на распознавание.
- `[ ]` **Оверлей переводчика**: Lock Mode (блокирующий) и Passthrough Mode (прозрачный для кликов).
- `[ ]` **UI "Бусины на нити"**: Рендеринг переведенных предложений с соединительными линиями, не перекрывая оригинал.
- `[ ]` **Yandex Translate API**: Подключение как fallback-переводчика.

## Этап 4: Дашборд и виджеты (Полировка)
- `[ ]` **Главный Dashboard (Home)**: Общая статистика, тумблеры виджетов, статусы подключений API.
- `[ ]` **Интерактивная история**: Блоки текста с иконками аудио, переключателями (оригинал/summary) и кнопкой копирования.
- `[ ]` **Виджет озвучки (Микро-плеер)**: Управление скоростью, выбор голосов, переключение языка на лету.
- `[ ]` **Контекстные иконки**: Всплывающий микрофон возле курсора в текстовых полях.
