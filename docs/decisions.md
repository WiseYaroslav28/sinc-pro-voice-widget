## Актуальные решения

### [ADR-0004] Синхронизация сохраненной скорости TTS с UI при запуске
* **Дата:** 2026-06-24
* **Статус:** `Active`
* **Категория:** UX
* **Сущность:** Виджет TTS (tts-widget.js)
* **Действие:** APPROVED
* **Контекст:** Исправление неверного отображения скорости (1.0x вместо сохраненного значения) на кнопке скорости воспроизведения при запуске приложения.
* **Решение:** При инициализации виджета tts-widget.js добавлен вызов функции updateSpeedUI(currentSpeed) сразу после ее объявления. Это гарантирует, что при первом запуске приложения скорость на кнопке панели управления и слайдер тонкой настройки синхронизируются с реальным сохраненным значением из localStorage.
* **Связанные файлы:** src/tts-widget.js

### [ADR-0003] Автоматический перезапуск микрофона после сна ОС
* **Дата:** 2026-06-24
* **Статус:** `Active`
* **Категория:** Architecture
* **Сущность:** Микрофон (capsule.html)
* **Действие:** APPROVED
* **Контекст:** Решение проблемы зависания диктовки/записи после долгого простоя или глубокого сна операционной системы.
* **Решение:** Перед каждым стартом записи в startRecordingProcess выполняется валидация закешированного MediaStream (проверка active и live-статуса треков). Если поток протух (например, после сна ОС), старые треки освобождаются через track.stop(), и поток инициализируется заново через getUserMedia.
* **Связанные файлы:** src/capsule.html

### [ADR-0002] Предотвращение принудительного показа капсулы в фоновом режиме
* **Дата:** 2026-06-24
* **Статус:** `Active`
* **Категория:** UX
* **Сущность:** Капсула (capsule.html)
* **Действие:** APPROVED
* **Контекст:** Исправление бага, при котором скрытая капсула выводилась на экран при скрытии главного дашборда или при переходе курсора на другой монитор.
* **Решение:** В функцию resizeToState добавлена проверка режима капсулы. Если режим равен 1 (скрыта, но работает) и состояние неактивно (idle/hover), вызовы позиционирования через Rust SetWindowPos блокируются, что предотвращает принудительное отображение окна на экране.
* **Связанные файлы:** src/capsule.html

### [ADR-0001] Паспорт проекта: SINC PRO Voice Widget
* **Дата:** 2026-06-24
* **Статус:** `Active`
* **Категория:** Architecture
* **Сущность:** Проект: SINC PRO
* **Действие:** APPROVED
* **Контекст:** Инициализация паспорта проекта при подключении Memory Engine.
