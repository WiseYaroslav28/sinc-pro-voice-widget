# Журнал изменений (Changelog) — SINC PRO

Все важные изменения в проекте голосового ассистента SINC PRO будут документироваться в этом файле.
Формат ведения журнала основан на принципах [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

---

## [2.0.2] — 2026-05-23

### Добавлено
- **Кнопка очистки текста `🧹`**: На верхнюю панель управления в режиме `full` добавлена кнопка быстрой очистки текстового поля. Она также сбрасывает текущее озвучивание.
- **Кастомное контекстное меню `ContextMenu`**: Создано стильное контекстное меню на customtkinter для замены стандартного `tk.Menu`, которое зависало на Windows в безрамочном режиме.
- **Умное позиционирование**: Контекстное меню автоматически определяет границы экрана и разворачивается вверх или влево, чтобы не выходить за пределы дисплея.
- **Автоматическое закрытие**: Контекстное меню закрывается по клику вне его области, по нажатию `Escape` или потере фокуса.
- **Интеграция правого клика**: Вызов меню привязан ко всем элементам приложения во всех режимах (Full, Mini, Micro), кроме текстового поля `CTkTextbox`.

### Исправлено
- **Плавный ресайз рамки**: Устранены прыжки рамки перевода вправо при изменении её размеров за правую и нижнюю границы за счет фиксации исходных координат окна перед началом перетаскивания.
- **Исправление автоперевода**: Устранена критическая ошибка `TypeError` при отпускании левой кнопки мыши в селекторе области (сигнатура коллбека `on_area_selected` обновлена для поддержки 5 аргументов).
- **Мгновенный перевод**: Изображение из селектора передается напрямую в рамку перевода через метод `translate_precropped`, исключая мерцание экрана и повторный захват области.
- **Мгновенная пауза воспроизведения**: Воспроизведение звука останавливается немедленно при нажатии "Паузы" (MCI устройство глушится командой `stop`), а при возобновлении текущее предложение читается заново с начала.

## [2.0.1] — 2026-05-22

### Добавлено
- **Селектор области экрана `AreaSelector`**: При запуске переводчика экран затемняется, и пользователь может выделить точную область экрана для перевода с помощью инструмента "ножницы".
- **Интеграция настроек хоткеев**: Возможность смены горячих клавиш озвучки (по умолчанию `ctrl+shift`) и перевода экрана в меню настроек.

### Исправлено
- **Конфликт горячих клавиш**: Хоткей переводчика экрана по умолчанию изменен с `ctrl+shift+t` на `ctrl+alt+t`, чтобы предотвратить конфликты с хоткеем озвучки `ctrl+shift`. Реализовано автоисправление старых настроек.
- **Ошибки WinRT при компиляции**: В скрытые импорты PyInstaller добавлены модули `winrt.windows.storage.streams` и `winrt.windows.foundation.collections`, что решило проблему падения распознавания текста (OCR) в собранной версии `.exe`.

## [2.0.0] — 2026-05-20

### Добавлено
- **Функция "Переводчик экрана"**:
  - Распознавание текста (OCR) на базе встроенного в Windows 10/11 WinRT API (высокая скорость и точность без интернета).
  - Пакетный перевод через `deep-translator` (Google Translate API).
  - Прозрачная рамка перевода `ScreenTranslatorFrame` с возможностью сквозных кликов (click-through) и наложением переведенных плашек поверх оригинального текста.
  - Кнопки управления на рамке: Перевести `🔄`, Показать/Скрыть оригинал `👁️`, Озвучить переведенное `🔊` (с интеграцией в основное окно) и Закрыть `✕`.
  - Кнопка вызова переводчика экрана на верхней панели с динамически генерируемой Pillow-иконкой `⛶ A文`.
- **Редизайн кнопок управления режимами**: Иконки переключения размеров виджета заменены на более понятные (`⤢` для Full, `▬` для Mini, `●` для Micro).

## [1.0.0] — 2026-05-15

### Добавлено
- Базовый функционал голосового ассистента SINC PRO: озвучка выделенного текста в любом приложении по нажатию `ctrl+shift`.
- Интеграция Edge TTS (высококачественные нейросети Microsoft) с поддержкой русского, английского и других языков.
- Три режима отображения:
  - **Full**: Полный редактор текста с подсветкой читаемых предложений и прогресс-баром.
  - **Mini**: Компактная плеер-панель для управления скоростью и воспроизведением.
  - **Micro**: Микро-кружок с базовыми кнопками воспроизведения и остановки.
- Сохранение настроек голоса, скорости и оверлеев в `voice_settings.json`.
